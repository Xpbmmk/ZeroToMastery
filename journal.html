2019.11.23

Learn command language used in Terminal to sync my local file with Github.


Git clone [+URL from GitHub repository]: clone the gidhub file to local file. Usually set up the first time work on new GitHub repository.
Git add [+file name] : update modified files Tip: ‘ git add . ‘ can update all modified files.
Git commit -m [+message or action name] : message or action name will appear at Github
Git push : push changes from local file to Github.
Git pull : pull changes from GitHub to local file.
Git branch [+branch name] : create new branch.
Git checkout [+branch name]: check the status of that branch.
Git merge master : merge branch to master branch.
